Email Editor

Email Editor - редактор email-писем. Ссылка на код - "Перейти".

Редактор писем

Пользователи могут писать и редактировать текст писем, менять стиль текста на жирный, курсив или подчёркнутый. Также есть возможность отправлять письма(на сервер). Отправленные письма появляются нижу.

Используемые технологии

Ниже представлены технологии, используемые для данного проекта.

React.js

Пользовательская часть написана на React.js

SCSS

Стили компонентов написаны на SCSS

Lucide-React

Для создания иконок использовался Lucide-React.

TypeScript

Проект написан на TypeScript.

Axios и React-Query

Для взаимодействия с JSON-БД были использованы Axios и React-Query.

JSON-server

JSON-server используется для хостинга БД.
