React Practice

React Practice - проект, демонстрирующий использование достаточно большого функционала React. Код - "Перейти".

Описание

В данном проекте реализованы: Авторизация, роутинг, приватные и публичные маршруты, пагинация и динамическая пагинация, бесконечная лента, анимации, работа с API, Loader и мемоизация, кастомные хуки и глобальные данные(Context), создание и удаление объектов, фильтрация и поиск.

Используемые технологии

Ниже представлены некоторые технологии и функционал React, используемые для данного проекта.

React.js

Приложение написано на React.js

CSS

Стили компонентов написаны на CSS

npm

Пакетным менеджером был выбран npm

Анимация

Для создания анимация была использована библиотека react-transition-group.

Мемоизация

В проекте реализована мемоизация.

Кастомные хуки

В проекте используются кастомные хуки для ожидания загрузки информации(Loader), бесконечной ленты постов, и сортировки списка постов.

API и axios

Для взаимодействия с API(Json Placeholder) был использован axios.

Loader

В приложении реализован Loader, отвечающий за предоставление информации только после её полной загрузки.

Пагинация и динамическая пагинация

В проекте реализована как обычная пагинация, так и динамическая пагинация(бесконечная лента), с помощью Observer.

Авторизация (Context)

В проекте реализована система авторизации с помощью Context(useContext).

Роутинг

В проекте реализованы приватные и публичные маршруты, для авторизованных пользователей и гостей соответственно. Использовалась библиотека react-route-dom v6.
