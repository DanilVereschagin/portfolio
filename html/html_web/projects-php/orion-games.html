Orion Games

Orion Games - онлайн магазин игр. Ссылка на код - "Перейти".

Пользовательская часть сайта

Пользователи могут покупать, добавлять в библиотеку, скачивать и комментировать игры. Также есть страницы личного кабинета и библиотеки игр. На главной странице выводится список всех пользователей.

Админская часть сайта

Администратор может изменять данные пользователей и удалять их, а также добавлять и удалять новые игры и компании. В том числе ему доступна возможность совершить email-рассылку всем пользователям.

Используемые технологии

Ниже представлены технологии, используемые для данного проекта. В данном проекте не были использованы серверные фреймворки.

HTML5, CSS3 и Bootstrap

Для пользовательской части сайта использовались html5, css3 и bootstrap.

Шаблоны

Были использованы бесплатные шаблоны youplay, которые в дальнейшем были переработаны под данный проект.

PHP

Серверная часть написана на чистом PHP(включая библиотеки).

MySQL и PDO

Как база данных использовалась MeSQL. Для взаимодействия с сервером использовалась библиотека PDO.

Nginx

Сайт стоял на веб-cервере Nginx.

Redis(кэш и сессии)

В данном проекте реализованы сисетмы сессий и кэширования с использованием Redis.

DI

В данном проекте реализована своя система DI(внедрение зависимостей)

XSS и CRLF защиты

В данном проекте реализованы защиты от XSS(распространенный тип веб-атаки, заключающийся во внедрении на страницу сайта или приложения вредоносного кода) и CRLF(уязвимость, позволяющая злоумышленнику внедрить символы перевода строки в запрос, тем самым изменив логику работы программы) атак.

Mail

Реализована возможность email-рассылки пользователям.

API

Данный проект получил реализацию своих собственный API(при постановки на глобальный хостинг есть возможность получать с него информацию со сторонних сайтов). Реализовано взаимодействие с API Стима(Steam). На интерфейсе магазина игры из Стима имеют чёрный фон, а не прозрачный.

CLI

Реализована система автоматических действий через cron(сбор информации, отправка рассылок, обновление закэшированных данных).
