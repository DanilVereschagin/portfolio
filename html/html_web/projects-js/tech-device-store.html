Tech Device Store

Tech Device Store - онлайн магазин техники. Ссылка на код - "Перейти".

Вид магазина для Гостя

Вид магазина для гостей. Пользователь может перейти на страницу товара, регистрации или авторизации.

Страница регистрации

Пользователь может зарегистрироваться на сайте. Он должен ввести логин и пароль для создания учетной записи. С этой страницы можно перейти на страницу авторизации.

Страница авторизации

Пользователь может авторизоваться на сайте. Он должен ввести логин и пароль для доступа к сайту. После этого он будет перенаправлен на страницу магазина.

Страница магазина для авторизованного пользователя

На данной странице пользователь может сортировать товары по типу и бренду, а также перейти на страницу товара, нажав на него.

Страница магазина для администратора

То же самое, что и для авторизованного пользователя, но также есть кнопка перехода в панель администратора в навигации.

Страница устройства

На данной странице авторизованный пользователь может посмотреть информацию о товаре, его свойства и рейтинг, а также сам поставить оценку товару или добавить его в корзину.

Страница корзины

На данной странице авторизованный пользователь может посмотреть список товаров, которые он добавил в корзину, и убрать ненужные, а также купить всё, что есть в есть в корзине. Над кнопкой оформления заказа видна сумма всех товаров в корзине.

Страница панели администратора

На данной странице администратор может создавать новые товары, бренды и типы.

Окно создания типа

В данном окне администратор может создать новый тип.

Окно создания бренда

В данном окне администратор может создать новый бренд.

Окно добавления товара

В данном окне администратор может добавить новый товар, выбрать его тип и бренд, а также добавить свойства.

Используемые технологии

Ниже представлены технологии, используемые для данного проекта.

HTML5, CSS3 и React-Bootstrap

Для пользовательской части сайта использовались html, кастомный css и react-bootstrap.

JS

Весь код проекта был написан на JS.

PostgreSQL и Sequelize

Как база данных использовалась MeSQL. Для взаимодействия с БД на сервере использовалась ORM - Sequelize.

React.js

Пользовательская часть проекта реализована с помощью библиотеки React.js

Node.js

Серверная часть проекта реализована с помощью Node.js

ExpressJS

Веб-фреймворк для серверной части проекта. Используется для реализации получения запросов от клиентской части приложения к серверу.

Bcrypt

Для хеширования паролей использовалась библиотека Bcrypt.

Axios

Для отправки запросов на сервер используется библиотека Axios.

Навигация

Для реализации навигации использовалась библиотека react-router-dom v6.

MobX

Для хранения состояния проекта использовалась библиотека MobX.

JWT-токены

Для работы с токенами доступа использовалась библиотека JWT.
