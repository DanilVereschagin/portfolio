Flower Store

Flower Store - серверная часть магазина цветов. Ссылка на код - "Перейти".

Магазин цветов

Есть возможность добавлять новые цветы в ассортимент и получать список всех цветов. Используются несколько способов обращения к серверу: REST API, GraphQL.

Используемые технологии

Ниже представлены технологии, используемые для данного проекта.

Nest.js

Серверная часть написана на Nest.js

Yarn

Пакетный менеджер

TypeScript

Проект написан на TypeScript.

PostgreSQL

В проекте используется БД PostgreSQL.

Prisma

Для взаимодействия с БД и кэшированием запросов использовалась Prisma.

Валидация

В проекте присутствует валидация данных.

Микросервисы

В данном проекте присутствует применение микросервисов.

GraphQL

Совместно с REST API используется язык запросов GraphQL.

Postman

Для удобного управления запросами к серверу использовался Postman.

WebSockets

В проекте реализовано постоянное соединение через Socket.io

Тестирование

В данном проекте проводилось модульное, а также End-to-End тестирование с помощью библиотеки Jest.
